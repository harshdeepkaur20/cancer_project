# Cancer Analysis & Prediction Dashboard

An interactive dashboard built using Python, Streamlit, and Machine Learning for analysis and prediction of multiple cancer types including Breast Cancer, Lung Cancer, Skin Cancer, and Blood Cancer.

## Features
- Interactive cancer data analysis dashboard
- Machine Learning based cancer prediction
- Random Forest Classifier implementation
- Dynamic visualizations using Plotly
- Support for multiple cancer datasets
- User-friendly Streamlit interface

## Tech Stack
- Python
- Streamlit
- Pandas
- NumPy
- Scikit-learn
- Plotly

## How to Run
```bash
pip install -r requirements.txt
streamlit run cancerproj.py
