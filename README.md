# Cancer Analysis & Prediction Dashboard

## Overview

The Cancer Analysis & Prediction Dashboard is an interactive web application built using Python, Streamlit, and Machine Learning. The project enables users to analyze cancer-related datasets and predict cancer outcomes using a Random Forest Classifier.

The dashboard supports multiple cancer types, including Breast Cancer, Lung Cancer, Skin Cancer, and Blood Cancer, and provides visual insights through interactive charts and graphs.

## Features

* Interactive Cancer Data Analysis Dashboard
* Cancer Prediction using Machine Learning
* Random Forest Classifier Model
* Support for Multiple Cancer Datasets
* Dynamic Visualizations using Plotly
* User-Friendly Streamlit Interface
* Real-Time Prediction Results

## Tech Stack

* Python
* Streamlit
* Pandas
* NumPy
* Scikit-learn
* Plotly
